# Daniel Passos' Blog

This site is built using [Jekyll](https://jekyllrb.com/), a static site generator.

## Prerequisites

This site requires Ruby and Node.js to be installed on your system.

### Ruby

We recommend using [rbenv](https://github.com/rbenv/rbenv) to manage Ruby versions.

1. Install rbenv following the [official installation guide](https://github.com/rbenv/rbenv#installation)
2. Install the required Ruby version:
   ```bash
   rbenv install
   ```

### Node.js

We recommend using [nvm](https://github.com/nvm-sh/nvm) to manage Node.js versions.

1. Install nvm following the [official installation guide](https://github.com/nvm-sh/nvm#installing-and-updating)
2. Install the required Node.js version:
   ```bash
   nvm install
   nvm use
   ```

## Development

To start the development server, run:
```bash
bundle exec jekyll serve
   ```
